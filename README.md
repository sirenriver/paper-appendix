# paper-appendix
# 论文附录图片 / Paper Appendix Figures

本仓库包含本人学位论文《鸟类血孢子虫宿主扩张动态及其影响因素研究》的补充材料，提供论文正文及附录图表的高清版本。

This repository contains supplementary materials for the thesis titled "*TEMPORAL DYNAMIC AND FACTORS INFLUENCING HOST-RANGE EXPANSION OF AVIAN HAEMOSPORIDIAN*", providing high-resolution versions of all figures in the main text and appendices.

## 使用说明 / Usage Instructions

2. 点击右侧的 **"Download"** 或 **"View raw"** 即可下载
3. 解压后获得全部高清图片
